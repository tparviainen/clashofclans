Clash of Clans

.NET Standard library for Clash of Clans API

Info

This repository contains a .NET Standard library for accessing Supercell's Clash of Clans API. In order to use the functionality provided by this library you need an API key (token) that can be created in Clash of Clans developer web site.

100% Clash of Clans API coverage, including the December 2019 Update.
